HTML - стандартизированный язык разметки документов для просмотра веб-страниц в браузере.

Возможности HTML:

  • Форматирование текста: выделение курсивом, жирным шрифтом, подчёркивание, изменение размера шрифта и создание нумерованных или маркированных списков.

  • Текстовые блоки: создание заголовков разных уровней, абзацев и переносов на новую строку.

  • Таблицы: создание таблиц с разным количеством строк и столбцов, установка фиксированной высоты и ширины таблицы, добавление заголовков к столбцам.

  • Вставка объектов: встраивание различных объектов, таких как изображения, звуковые и видеофайлы, текстовые документы и другие мультимедийные элементы на веб-страницу.

  • Гиперссылки: создание гиперссылок, которые позволяют пользователям переходить к другим страницам или открывать файлы.

  • Создание простых форм: создание простых форм для ввода текстовой информации, выбора пунктов из списка и отправки данных на сервер.

CSS — это формальный язык описания внешнего вида страницы, каскадные таблицы стилей.

Возможности CSS включают:

  • Оформление HTML-элементов (размеры, отступы, цвет фона, тени, скругления);

  • Построение сеток для расположения контента;

  • Создание анимаций без использования JavaScript;

  • Создание адаптивного дизайна (изменение отображения в зависимости от размеров устройства, на котором просматривают сайт).

JavaScript — это интерпретируемый язык программирования, который используют для написания frontend- и backend-частей сайтов, а также мобильных приложений.

Возможности JS:

  • Открывать всплывающие окна;

  • Отправлять и принимать запросы с других ресурсов;

  • Манипулировать элементами страницы;

  • Подгружать контент;

  • Выполнять анимацию;

  • Управлять воспроизведением мультимедиа;

  • и т.д.
